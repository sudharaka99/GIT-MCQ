# GIT Grade 12 Complete MCQ Bank – 8 Units (550+ Questions)
This is a complete, standalone MCQ (Multiple Choice Question) examination system for General Information Technology (GIT) Grade 12 students in Sri Lanka. The entire question bank is based on the NIE Teacher's Guide (Effective from 2017) and covers all 8 competency levels of the syllabus.

Each unit is a self-contained HTML file that works offline, requires no internet connection (after download), and provides instant feedback with visual scoring.
